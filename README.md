# ETL-Deteccion-Fraude
# 🛡️ Fraud Detection Pipeline (ETL + Machine Learning)

## 📌 Descripción
Pipeline de datos completo para detección de fraude en transacciones financieras, integrando ETL y un modelo de Machine Learning (Random Forest).

---

## 🏗️ Arquitectura

- Data (raw, processed, predictions)
- ETL
- Feature Engineering
- Model Training
- Prediction Pipeline

---

## 🔄 Flujo del sistema

Raw Data → ETL → Features → Model → Predictions

---

## ⚙️ Tecnologías

- Python
- Pandas
- Scikit-learn
- Random Forest
- Logging

---

## 📊 Resultados

- Accuracy: ~99%
- Recall (fraude): 0.84
- Precision (fraude): 0.52
